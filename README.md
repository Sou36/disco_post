disco_post2.py

→ターゲットをグループに入れて自分は抜けたい際に使用します。

---

### ～実行環境～

pythonのplaywright準備してから実行してください。

playwrightはCMDで
```cmd
pip install playwright
```

その後CMDで
```cmd
playwright install
```

すれば簡単に入れれます。過去に入れたことがあればしなくていいです。
